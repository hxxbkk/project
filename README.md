메인화면

평소 컴퓨터를 사용하면서 윈도우라는 운영체제가 저의 일상에 깊숙히 스며들어 있음을 느꼈습니다. 그래서 윈도우 기능을 구현한 웹 페이지를 만들면 어떨까 라는 생각을 하게 되었고 윈도우 하면 떠오르는 배경화면과 레이아웃을 적용해 평소 사용자들이 익숙함을 느꼈던 그대로를 구현하기 위해 노력했습니다.

- React
